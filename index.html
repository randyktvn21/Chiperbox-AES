Chiperbox

Chiperbox AES-128

Ruang simulasi kriptografi bertema secure vault untuk melihat proses AES-128 dalam bentuk kotak-kotak data. Setiap byte ditempatkan dalam State Box 4 × 4, lalu perubahan Key Expansion, SubBytes, ShiftRows, MixColumns, AddRoundKey, dan operasi inverse ditampilkan sebagai alur penguncian data.

Ruang Kerja
Vault Block
Algoritma
AES-128 · 10 Ronde
Field Byte
GF(28) / 0x11B
Isi Chiperbox
  • AES dijalankan langsung di browser dengan Vanilla JavaScript, tanpa CryptoJS, Web Crypto API, atau library kriptografi pihak ketiga.
  • Setiap ronde disajikan sebagai kartu State Box, sehingga perubahan byte bisa dibandingkan sebelum dan sesudah transformasi.
  • Disertai tabel referensi AES: S-Box, inverse S-Box, Rcon, MixColumns, serta InvMixColumns.

Box Input

Masukkan data uji atau gunakan sampel yang sudah dibuat untuk varian Chiperbox.

Mode hex membutuhkan tepat 32 digit. State dipetakan secara column-major sesuai AES: s[r,c] = input[r + 4c].

Kunci harus tepat 32 digit heksadesimal atau 16 byte.

Referensi Konstanta AES

Daftar nilai dasar yang dipakai dalam transformasi AES-128 dan inverse transform untuk dekripsi.

Catatan: Chiperbox dibuat sebagai media belajar AES-128 satu blok. Tema secure vault dipakai agar data, kunci, ronde, dan output terlihat seperti unit-unit box yang dapat diaudit satu per satu.
